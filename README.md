# 我的 n8n 自动化学习之旅 🚀

这是一个记录我从零开始学习 n8n 自动化工具的代码库。

## 📂 练习项目目录

### 1. Hello World (01_Hello_World.json)
- **功能**：基础练习，了解 n8n 界面。
- **学到的节点**：Schedule, Google Sheets。

### 2. Web3 AI 日报 (02_Web3_AI_News.json)
- **功能**：每天早上自动抓取 Coindesk 新闻，用 AI 写摘要并发送邮件。
- **核心技术**：RSS, OpenAI/Gemini, Email。

### 3. BTC 价格监控报警 (03_BTC_Price_Alert.json)
- **功能**：每小时监控比特币价格，跌破 80000 美元发邮件报警。
- **核心技术**：HTTP Request (API), If 逻辑判断。

### 4. 闪念笔记机器人 (04_Flash_Notes.json)
- **功能**：通过 Telegram 发送消费记录，AI 自动拆解并存入表格。
- **核心技术**：Telegram Trigger (Webhook), JSON 数据清洗。
